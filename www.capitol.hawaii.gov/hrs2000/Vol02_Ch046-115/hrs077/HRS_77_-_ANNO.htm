Note

Chapter repealed July 1, 2002. L 2000, c 253, §131. Effect on rules, ordinances, directives, rights, benefits, etc. L 2000, c 253, §§144 to 149.

This chapter is based upon L 1955, c 274, §2, which completely reenacted RL 1945, Chapter 3. Chapter 3 was based upon L Sp 1941, c 88, and was amended by L 1945, c 263, §§1, 3, 5, 6; L 1947, c 208, §§1, 2; L 1949, c 282, §1, c 298, §§2, 3; L 1951, c 21, §1, c 51, §3, c 319, §2, c 320, §1; L 1953, c 278, §2. Source notes following each section commence as of L 1955, c 274. See §76-6 making inoperative any provision which jeopardizes federal funds.

Privatization. Development and implementation of performance based budgeting and a managed process for public-private competition of government services. Authorization of State and counties to contract for necessary goods and services without being void as contrary to the laws on civil service, merit principles, and collective bargaining (repealed June 30, 2001). L 1998, c 230.

Workforce restructuring: voluntary severance benefits or special retirement incentive benefit (repealed June 30, 2003). L 2000, c 253, §§115 to 124.

 

Attorney General Opinions

Because the procedure outlined in §76-2 is limited in its application to interpretations of this chapter and chapter 76, the constitutionality of any provision of the two chapters is an issue beyond the bounds of that process. Att. Gen. Op. 97-6.
