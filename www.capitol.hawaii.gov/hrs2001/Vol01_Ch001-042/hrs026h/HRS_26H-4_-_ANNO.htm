Regulatory programs for professions and occupations that were subject to assessment and repeal under prior law. Prior to 1994, this section contained references to many regulatory chapters and sections scheduled for assessment and repeal. This information was noted in the main volumes at the specific profession or occupation chapter or section affected as "subject to reassessment, this chapter is repealed effective (date of scheduled repeal)". L 1994, c 279, §3 amended this section by removing all of the chapters, sections, and repeal dates rendering the main volume notes obsolete.

Revision Note

The source notes for this section begin with L 1994, c 279, §3 which replaced the contents of this section in its entirety. Prior source notes to this section are in the main volume.

Amendments prior to supersession by L 1994, c 279, §3 not referenced in the main volume are as follows: L 1994, c 39, §1, c 40, §1, c 183, §1, c 184, §2, c 185, §§1, 2, c 251, §3, and c 277, §2.
