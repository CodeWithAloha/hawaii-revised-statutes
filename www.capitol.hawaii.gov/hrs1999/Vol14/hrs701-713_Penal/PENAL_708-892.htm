[§708-892] Unauthorized computer use. (1) A person commits the offense of unauthorized computer use if the person intentionally and without authorization accesses, alters, injures, disrupts, damages, or destroys any computer, computer system, computer network, computer program, computer software, or any data stored therein.

(2) Unauthorized computer use is a class C felony. [L 1992, c 225, pt of §2]
